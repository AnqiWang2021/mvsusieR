Multivariate simple regression comparison with univariate¶

Here I use multivariate prior (not mash mixture) for msusie call, and compute result with susie.

Multicariate calculation is implemented in the BayesianMultivariateRegression class with an interface added to the main function such that the code will be triggered when prior variance is a matrix.
